# Cifra7
Um dever de casa para aulas de criptografia e segurança de sistemas da faculdade de Ciência da Computação.

Para executar você deve entrar no diretório do php e abrir o cmd. Depois disso é só usar php -S 0.0.0.0:8000

O site possui uma função para informar qual é o ip que deve ser usado para que outros computadores na mesma rede acessem o site através do seu próprio computador.
